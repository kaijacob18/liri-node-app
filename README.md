# liri-node-app

## Liri Bot Project
In this project, there are total of 4 command used to grab data from API listed below...
- Spotify
- OMDB
- Bands In Town
- random.txt (text file)
### The 4 Command Line used
1. spotify-this-song
2. movie-this
3. concert-this
4. do-what-it-says
